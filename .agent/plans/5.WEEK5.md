## Pages

### Chat (/)
- Message thread with user questions and AI answers
- Trust score badge on each answer (green ≥ 0.75, yellow 0.5–0.74, red < 0.5)
- Source citations (document title, source type, department)
- "Needs Review" indicator when flagged
- Loading skeleton while waiting for response
- Chat history persists when navigating between tabs (ChatContext)
- Clear button to reset conversation

### Ingest (/ingest)
- PDF drag-and-drop uploader — text extracted client-side via `pdfjs-dist`, no raw file sent to backend
- Manual form: document ID, title, source type (dropdown), department, text
- Add multiple documents in one session
- Success confirmation shows chunks created

### Review Queue (/review)
- List of pending reviews with trust score, question, answer
- Approve / Reject / Modify actions per item
- Modify opens inline editor for corrected answer
- Refresh button to reload queue

## Backend Change (CORS)

`src/api/main.py` — added `CORSMiddleware` allowing:
- `http://localhost:5173` (Vite dev server)
- `http://localhost:4173` (Vite preview)

Without this, every browser request from the frontend would be blocked.

## Running Locally

Terminal 1 — backend:
```bash
uvicorn src.api.main:app --reload --port 8000
```

Terminal 2 — frontend:
```bash
cd frontend && npm run dev
```

Open `http://localhost:5173`.

## API Base URL

Configured via `VITE_API_URL` env var (defaults to `http://localhost:8000` via Vite proxy).
For AWS production: set `VITE_API_URL` to the ALB DNS name at build time.
